# About 2025/26 CSC 4505: Computer Graphics and Visual Computing

2025/26 CSC 4505 is an undergraduate-level course offered in the [Department of Computing and Informatics](http://cs.unza.zm) [1] at [The University of Zambia](http://unza.zm) [2]. CSC 4505 introduces students to the various techniques used to model 2D and 3D objects and environments. The course focuses on the mathematical and programmatic foundations required to present graphical information effectively, ensuring that complex aspects of the digital world can be easily visualised.

## Learning Outcomes

* Explain the principles underlying the operation of graphical input/output devices.
* Model and create 2D and 3D images using standard graphics APIs (OpenGL).
* Implement primitive operations for the representation of 2D lines, curves, and filled polygons.
* Use linear algebra techniques for affine and perspective transformations on points and objects.
* Develop graphical applications using the Python programming language and OpenGL library.
* Compare algorithms for visible surface determination and photorealistic rendering.

## Repository Structure

This repository is organised into the following directories:

* **`code/`**: Source code examples and implementations of graphics primitives.
* **`slides/`**: Lecture slide decks covering theoretical concepts.
* **`labs/`**: Lab problem sets and practical exercises.
